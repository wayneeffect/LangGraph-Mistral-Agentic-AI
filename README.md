# Agentic AI: Multi-Agent Research & Code Generator

A LangGraph-based multi-agent system using Mistral AI for parallel research, code generation, and validation.

## Features
- Parallel agents: Research, Code Gen, Validator
- Free Mistral API integration
- Deployable on Render

## Setup
1. Get free Mistral API key: mistral.ai
2. `pip install -r requirements.txt`
3. `streamlit run app.py`
